这份建议我喜欢并支持

发表于 2020-05-23 | 分类于 观点时评 |

看到了新京报的一则对刘小兵的《关于自闭症群体面临的困境与对策建议》的报道，觉着挺好，这个建议我喜欢并支持。

刘小兵的建议

新京报快讯（记者 倪伟）“应该将自闭症从精神残疾中分离出来，作为一项单独的残疾类别。在此基础上制定更有针对性的政策，保障自闭症患者及其家庭的权益。”2020年全国两会期间，全国人大代表、上海财经大学公共经济与管理学院院长刘小兵带来一份《关于自闭症群体面临的困境与对策建议》。

这份建议建立在对自闭症群体现状的调研基础上。刘小兵调研发现，国家层面与地方层面都没有针对自闭症患者的普查。社会关注度缺乏，导致针对自闭症家庭的社会服务和相应保障严重缺乏，一些制度规定不合理，设施、服务以及专业人员等方面供给不足且增长缓慢。

例如，具有托养功能的特殊学校需要按照户籍安排入学，如果自闭症患者住所和户籍地不一致，即使学校有名额，此患者也无法就近入学。自闭症患者大多伴随其他疾病，且语言障碍导致其无法准确表达身体不适，需要专业的精细化照护，但自闭症照护就业门槛低，相关人员缺乏专业的照护知识和技能。另外，公众缺乏认知，导致自闭症患者遭受不同程度歧视。

同时，国家针对残障人士制定的政策法规不适用于自闭症群体。与传统意义上的精神残疾不同，自闭症的成因及表现较为复杂，难以通过药物治疗。其核心障碍在于语言障碍和社交功能缺失，不能简单定义为精神残疾。

刘小兵认为，解决自闭症群体的困境，首先需要重新界定自闭症患者。他在议案中建议，完善政策法规体系，将自闭症从精神残疾中分离出来，作为一项单独的残疾类别。在此基础上制定更有针对性的政策，保障自闭症患者及其家庭的权益，可以参考英美等国将自闭症作为单独类别出台相应政策的做法。

同时，他建议开展自闭症患者全面摸底调查，包含自闭症患者的数量、性别、年龄构成、病情、区域分布等基本信息，合理规划下一步托管机构的数量和分布。通过普查数据，也能了解自闭症现状，提高社会对自闭症的认知程度，呼吁社会各界关注自闭症群体。

在提升自闭症群体托管机构建设方面，他强调要构建成年自闭症托养机构。自闭症托养机构具有一定的公益性质，应当由政府主导建立，打破户籍限制，安排就近接纳。区政府可提供配套资金，协助托养机构落地，完善相应的配套措施，支持并监督机构的运营。

“我们在调研中发现，自闭症家庭呼吁政府起主导作用，且承担一部分日间托管机构的服务费用，希望尽快建立照护专业度高、有政府补贴、邻近且便利的日间托管机构。”他说。

另外，他提出，应针对自闭症照护人员开展专业培训，加强考核，认证其从业资格，规范照护人员队伍的建设。在高等职业院校中，可以增加自闭症护理课程，加强相关专业学科建设。

在支撑力量方面，他提出相关部门要建立联动机制，动员社会力量参与，托养机构可以采取公建民营等模式。加强与医疗机构之间联动，如提供就医的绿色通道、委派医护人员驻点或定期到机构进行检查和服务；建立与公安部门之间的联动，保护自闭症患者的人身安全。同时，呼吁社会各界力量参与。

刘小兵提出来的这个算是代表建议，还不是代表提案。

提案和建议的区别

那人大代表提案和建议的区别是什么呢？我在上海市政府官网上找到了一则说明，我觉着这个解释应该比较权威了：

议案与代表建议的区别在于：

  1. 议案须由法律规定的单位以及一个代表团或者达到法定人数的代表共同提出，而代表建议一般是由一名代表或数名代表联名提出。
  2. 议案须经人民代表大会主席团或人大常委会审议通过方可立案，而代表建议则由人大常委会的办事机构交有关机关或组织研究处理。
  3. 议案涉及的内容必须是属于人民代表大会职权范围内的事，而代表建议可以不限于这个范围。
  4. 议案通过后具有法律约束力，承办机关必须执行，而代表建议可由承办单位根据条件，区别情况，研究办理。
  5. 人大代表在大会期间提议案有截止日期，超过规定时限的，不作议案而作为代表建议处理，而代表提出代表建议，则没有截止日期的规定。

向这些代表委员致敬

关于“自闭症”，我找了一下，之前其实已经有不少级别的人大代表或政协委员提出过相关的提案、议案、建议：

2007年，湖北省人大代表叶小缅的《关于自闭症儿童的康复教育问题》提案，建议政府应当专门为这些儿童开办矫治和教育的培训学校，或在聋哑学校、普通教育机构中设立特殊的教育班，为他们配备专门的辅导老师。

2012年，海南省政协台盟提案《关于加强自闭症（孤独症）儿童康复教育的建议》，海南省卫计委进行了回复。

2013年，全国人大代表、中华女子学院女性学系教授孙晓梅，呼吁政府加大投入为孤独症患者提供更多康复训练机会。

2014年，东港市人大代表李盛飞等《关于加强我市自闭症儿童救助工作的建议》

2015年，全国政协委员、民革党员周桐宇的《完善自闭症人士专业化、持续化关爱服务体系》的建议，提出制定自闭症诊断标准、建立自闭症早期筛查制度等6条建议。

2017年，全国人大代表张琼建议，将对自闭症儿童康复训练所需的康复经费纳入到医保的范畴，使这些孩子得到及时有效的治疗。

2017年，黑龙江省人大代表姜志梅的《关于加强孤独症康复教育与保障的建议》建议，黑龙江省教育厅进行了建议回复。

2017年，辽宁省人大代表王阳硕的《关于自闭症患者融合教育的建议》，沈阳市人民政府做了回复。

2018年，全国人大代表、内蒙古赤峰星之路自闭症儿童康复中心校长王欣会建议：自闭症儿童教育康复亟须法律保障。

2019年，全国人大代表、民革中央委员刘小兵提出一份《建立成年自闭症人士日托服务机构》的建言。

了解自闭症

什么是自闭症

有这样一群孩子：

不聋，却充耳不闻；

不盲，却视而不见；

不哑，却闭口不言……

他们被诗意地称作“星星的孩子”

维基百科：自闭症（英语：autism）为一种由脑部发育障碍所导致的疾病，其特征是情绪，言语和非言语的表达困难及社交互动障碍，会对限制性行为与重复性动作有明显的兴趣。家长一般会在自闭症孩童两到三岁时注意到其状况。其症状会渐渐加重，不过有些有自闭症的孩童在自闭症恶化之前，在一段时间内有着正常或接近正常的早期发展阶段成长里程碑，之后会出现一个或多个自闭症的特发特征，比如语言倒退等。

中国自闭症现状

《中国自闭症教育康复行业发展状况报告》显示，目前中国自闭症患病率约为1%，自闭症患者已超过1000万，0~14岁的儿童患者达200余万，这也就意味着，如果一个幼儿园有4个班级，每个班级有25个孩子，其中可能就有一个孩子是自闭症患儿。

由于社会对自闭症认识不足，很多时候，对自闭症的描述充满了污名化：“异常”“狭窄”“固执”。

联合国将将4月2日定为世界提高自闭症意识日。要提高社会认知，消除污名和歧视，帮助改善自闭症人士的生活质量，这样他们才能作为社会的一个组成部分过上充实而有意义的生活。

自闭症的五个早期行为标志

自闭症的五个早期行为标志，分别是

不看少看、不应少应、不指少指、不语少语和不当。

这些标志性行为很多时候在2岁前就已经出现。如果宝宝出现了这些行为，特别是同时具有这些行为，我们就应该非常警觉了。

不看或少看

不看或少看，是说目光接触异常。对人，特别是对人的眼部注视较少。脑部扫描也发现，和普通孩子相比，自闭症孩子看到玩具图片时，大脑更活跃，看到人脸图片，甚至是妈妈的脸，都不怎么活跃。

不应或少应

不应或少应，包括两项，一个是你叫宝宝的名字，他没有反应；就是所谓的没有呼名反应。还有一个是，宝宝不能跟别人产生共同注意，也就是说，不能和其他人关注同一个物体或事件。比如，你指着洋娃娃说：“宝宝，看，娃娃。”如果宝宝顺着你的手指，跟你同时看向洋娃娃，这就叫共同注意，共同注意在语言和社交技能的发展上非常重要。

不指或少指

不指或少指，是说缺乏恰当的肢体动作，不会对感兴趣的东西提出请求。不会摇头表示不要，不会有目的地用手指。

不语或少语

不语或少语，特指语言发展方面的延迟。语言理解和表达能力低下，常用错“你”“我”“他”。不愿与其他小朋友一起玩耍，对亲人无动于衷。

不当

不当，这是指使用物品异常，相关的感知觉异常，语言发展倒退。比如有刻板的重复动作，比如只玩某一个玩具，玩很长时间，玩的时候不理人，而且兴趣狭窄。对疼痛、寒冷反应迟钝，对危险没有恐惧感，时常自伤。

如果对照完这些标志行为，你发现自己的孩子符合，特别是符合多种这样的异常行为，你就应该警觉了。需要到医疗机构去做筛查，可以通过系统测试，做出早期筛查和干预指导。

8个防治措施

  1. 加强围产期卫生保健，实行优生优育。
  2. 营造和谐温馨的家庭气氛，多带孩子参加户外活动、接触大自然。
  3. 在日常生活中，向孩子灌输正确的人生观，使他明白生命的可贵之处。
  4. 培养孩子的听说能力，改善他们的语言功能。
  5. 对孩子的尝试，家长不应过早干预，应给予恰当的鼓励；动员孩子帮忙做家务；选择容易掌握的竞技活动，让孩子感受经历失败后成功的喜悦。
  6. 伤心时不避孩子，让孩子了解家长的情绪，同时也可示范宣泄情绪的方法。
  7. 当孩子遇到问题向家长倾诉时，不妨将以往的经历、挫折、失败和痛苦、所得的教训等，一一坦诚地讲给孩子听。
  8. 带孩子进行心理咨询，听听心理医生的建议。

我们能为自闭症儿童做些什么

作为教师：

  1. 运用视觉策略以配合学生对视觉信息有较强的反应及理解能力；

  2. 尽早建立有组织、有规律的学习环境；

  3. 尽早协助学生建立适切及有效的沟通技巧；

  4. 应自闭症学生所执著的一些特定主题，顺势引导，作为学习的动力；

  5. 提供有利的学习环境，减少学生因噪音、刺光、室温过高等而影响学习；

  6. 利用多种感官协助学生学习，避免一连串长的口头指令。

作为家长：

  1. 加深认识自闭症的真相，知己知彼，百战百胜；
  2. 确立信念，坚信孩子受训后，会比现在进步；

  3. 采取正面态度，面对培育孩子的挑战；

  4. 肯定孩子的强项和长处，并发掘及栽培他们的特殊天分，使他们尽展所长，贡献社会；

  5. 与学校及其他专业人士紧密合作；

  6. 建立支援网络，争取家人体谅、亲友支持和社区人士的接纳。

作为朋辈：

  1. 担起模范作用，协助自闭症同学投入课堂学习和活动；

  2. 透过现实情境中的社交活动，如集体游戏、茶会等，示范和指引自闭症同学学习和应用社交技巧；

  3. 本着接纳和支援的态度，使自闭症同学提高与人进行社交往来的意欲。

作为公众人士：

  1. 加深对自闭症的认识，体谅和支持自闭症儿童家长所面对的困难及挑战，避免因误解而怪责家长；

  2. 接纳和尊重自闭症人士，不要歧视或戏弄他们；

  3. 提供机会让自闭症人士融入社会，使他们能跟别人接触及沟通，克服自闭症带来的障碍和社会适应上的困难。

他们生而不同，但是他们敢于追逐梦想，打破平凡。让我们多一点理解与包容，多一点支持与关爱，让这些“星星之花”尽情绽放！
